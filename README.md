# MojoMath

MojoMath is mathematic library for Mojo-language.
And this library is under developing.

# How to install

To use this library, surely, user must install Mojo.
You can search the way to install Mojo-library on web.
Please search, for example, "Mojo install" on Google.

To install MojoMath in your computer, you can use ``make`` command.
If installation successes, then you can see "mojo_math.mojopkg" is gernerated.
After this, you can use MojoMath.
